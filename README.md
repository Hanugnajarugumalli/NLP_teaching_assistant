# NLP Teaching Assistant

Teaching assistant project for Natural Language Processing, including visualizations, summarization, and YouTube content processing.

## Setup

1. Create a virtual environment and install dependencies:

   ```bash
   pip install -r code/code/requirements.txt
   ```

2. Copy `code/code/.env.example` to `code/code/.env` and add your API keys (e.g. `REPLICATE_API_TOKEN`).

## Structure

- **code/code/** – Notebooks and scripts (TextBlob visualizations, summarizer, YouTube scraping)
- **output/** – Generated images and outputs
